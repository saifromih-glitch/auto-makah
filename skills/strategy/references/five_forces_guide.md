# 🧭 دليل تحليل القوى الخمس — Porter's Five Forces Guide

## القوة ١: تهديد الداخلين الجدد (Threat of New Entrants)

**تقييم ١-٥ (١=ضعيف، ٥=قوي):**

- [ ] حواجز الدخول مرتفعة؟ (رأس مال، تراخيص، تقنية)
- [ ] وفورات الحجم متاحة للمنافسين الحاليين؟
- [ ] العملاء مرتبطون بالعلامات الحالية؟
- [ ] القنوات التوزيعية محتكرة؟
- [ ] السياسات الحكومية تحمي السوق؟

## القوة ٢: قوة الموردين (Bargaining Power of Suppliers)

- [ ] عدد الموردين محدود؟
- [ ] تكلفة تبديل المورد مرتفعة؟
- [ ] المورد يمكنه التكامل الأمامي؟
- [ ] المنتج فريد/متخصص؟
- [ ] لا توجد بدائل للمواد الخام؟

## القوة ٣: قوة المشترين (Bargaining Power of Buyers)

- [ ] عدد المشترين قليل؟
- [ ] تكلفة التبديل منخفضة للمشتري؟
- [ ] المشتري حساس للسعر؟
- [ ] المنتج غير مميز؟
- [ ] المشتري يمكنه التكامل الخلفي؟

## القوة ٤: تهديد البدائل (Threat of Substitutes)

- [ ] بدائل أرخص متاحة؟
- [ ] أداء البديل مشابه أو أفضل؟
- [ ] تكلفة التحول للبديل منخفضة؟
- [ ] البديل من صناعة مختلفة تماماً؟

## القوة ٥: التنافس في الصناعة (Industry Rivalry)

- [ ] عدد المنافسين كبير؟
- [ ] نمو الصناعة بطيء؟
- [ ] تكاليف ثابتة مرتفعة؟
- [ ] حواجز الخروج مرتفعة؟
- [ ] المنتجات متشابهة (سلعة)؟
